    关键字所有字母都是小写

    在java中，数据类型分为基本数据类型和引用数据类型，其中基本数据类型有八种，其余全是引用数据类型。

    八种基本数据类型

    整数类型
    byte 一个字节=8bit（-128~127，声明byte赋值不能超过这个范围）
    short 2字节 (-215~215-1) int i = 1
    int 4字节 (-231~231-1) short s = 1
    long 8字节 (-263~263-1) long l = 3l (赋值后加l表示long类型)

    浮点类型
    double d = 1.22
    float f = 1.22f

    字符类型：用英文单引号括起来的单个的字母数字符号
    char c1 = 'c'
    char c2 = '1'
    char c3 = '%'
    char c4 = '\n'
    char c5 = ''

    **布尔类型（正负） **
    boolean b1 = true
    boolean b2 = false

    引用数据类型

    字符串：由多个字母数字符号共同组成的一个串，用双引号括起来
    String str == "Hello world";
    System.out.println(str);
    String是引用类型，可以使用null作为值

    String s0 = "Hello";
    String s1 = "Hello";
    String s2 = "He" + "ll" + "o"; 在内存中只存在一个“Hello”
    假设"Hello"的内存地址为xxxx，当声明s0变量时给s0变量赋值"Hello"，实际是让s0变量引用“hello”的内存地址xxxx;当再声明s1时同样引用同一个地址xxxx；而int i0=1和int i1=1 则会在内存中储存两个1的值

    char类型数据与数字进行计算时，转化为相应ASCII码进行计算

    整数\整数 舍弃小数部分
    int = 0;
    int k = i++：k=0,i=1
    int k = ++i: i=1,K=1

    字符串的+=是拼接
